Key Settings
Pilih aksi dari Picker di bawah.
Macro Editor — Urutan keystroke + delay. Pilih type, isi value, klik "+ Add". Pilih "Virtual Action" untuk assign fungsionalitas khusus dari picker di bawah.

Macros

Tap-Dance — 1 tombol, 4 fungsi. Klik slot (hingga menyala ungu), lalu pilih aksi dari Virtual Keyboard di bawah.

Tap-Dance Entries

Combos — Tekan 2–4 tombol bersamaan → aksi khusus. Klik slot K1–K4 atau Output, lalu pilih input langsung dari Virtual Keyboard di bawah (tanpa perlu modal).

Combo Entries

Alt Repeat Key — Tombol bergantung pada key terakhir. Contoh: Ctrl+Z → Alt Repeat jadi Ctrl+Y. Tekan tombol fisik langsung di field Last Key / Alt Key.

Alt Repeat Rules

MIDI Note Configuration (C2–C4) — Atur properti kustom pada note (seperti Custom Name, Override Velocity, & Transpose). Pilih note di Grid untuk mengeditnya.
MIDI CC Configuration — Konfigurasi Mode (Momentary/Toggle), Off/On Value dan Assign Control Change (CC).

Control Change (CC)

LED WS2811 — Konfigurasi strip LED. Mode Responsive menyalakan LED saat tombol ditekan.
